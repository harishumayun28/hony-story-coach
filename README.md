# HONY Story Coach


---

## What is this?

Humans of New York (@humansofnewyork) went silent on Tumblr in mid-2024 after 14 years and 8,289 posts, while continuing to publish the same long-form narrative content daily on Facebook and Instagram. Tumblr is a tag-driven discovery platform where cross-posting costs nothing — yet HONY used **zero tags** across their entire archive.

This project delivers a **pre-publish engagement advisor** called the **Story Coach** — a Streamlit app where Brandon Stanton pastes a draft story caption and receives:
- An engagement prediction (high / medium / low) based on patterns in 2,000 HONY posts
- Three plain-English editorial suggestions (e.g. "Consider shortening this story — posts under 400 words earn 40% more engagement")
- Tumblr tag recommendations to improve content discovery
- The story's predicted narrative theme based on LDA topic modeling

## NLP Methods Used

| Method | Section | Purpose |
|--------|---------|---------|
| TF-IDF Vectorization | Classification | Text representation for classifiers |
| VADER Sentiment Analysis | EDA + Features | Compound, positive, negative, neutral scores |
| Sentiment Arc Analysis | EDA | Intra-post emotional trajectory by engagement tier |
| Named Entity Recognition (NLTK) | EDA + Features | Entity count as engagement signal |
| LDA Topic Modeling | Topic Analysis | K=8 narrative themes, engagement by theme |
| N-gram Discriminative Analysis | NLP Analysis | Unigrams and bigrams that predict engagement class |
| Binary Classification | Modeling | Naive Bayes, Logistic Regression, RF, SVC, XGBoost |
| Regression | Modeling | log(notes) prediction with RF and XGBoost |
| SHAP Explanations | Dashboard | Per-post feature attribution — "why this score?" |
| Cosine Similarity Tag Recommender | Dashboard | Tag suggestions from similar high-engagement posts |

## Key Findings

**1. Standalone personal narratives earn 3–4× more engagement than serialized content.**
Topic modeling surfaces 8 narrative themes. Personal Struggle & Night Stories (0.94 notes/day) outperform serialized series like the Iran/Shahnameh arc (0.23 notes/day) by 4x.

**2. Shorter posts outperform longer ones.**
Posts over 600 words earn roughly one-fifth the engagement of posts in the 100–400 word range.

**3. HONY used zero tags across 2,000 posts.**
Tumblr is a tag-driven discovery platform. Every HONY post during the active era was invisible to anyone not already following the account — the single most actionable gap in the re-entry strategy.

**4. Engagement is predictable from content alone.**
Random Forest (Combined TF-IDF + engineered features): **cross-validated ROC AUC = 0.853 ± 0.026** — predicting high vs low engagement using only text features, no temporal signals.

## Model Details

### Target Variable
`notes_per_day = note_count / days_since_post` — normalizes for the fact that older posts have had more time to accumulate notes. Raw note_count correlates with year at r=−0.612.

### Two-Model Architecture
| Model | Purpose | AUC |
|-------|---------|-----|
| Random Forest (Combined) | Report metrics | 0.853 CV |
| XGBoost (Engineered features) | Dashboard SHAP explanations | 0.781 CV |

The performance model uses TF-IDF + 12 engineered features. The dashboard model uses engineered features only because their SHAP values map to human-readable content levers (word count, sentiment, dialogue ratio) that Brandon can act on.

### 12 Content Features
`word_count`, `sentence_count`, `avg_sentence_length`, `vader_compound`, `vader_pos`, `vader_neg`, `vader_neu`, `question_count`, `exclaim_count`, `dialogue_ratio`, `first_person_ratio`, `uppercase_ratio`

**Excluded:** `year`, `month`, `hour`, `dayofweek` — temporal confounds removed intentionally.

## Limitations

- **Image modeling is out of scope.** HONY's visual style is highly uniform, so text variance explains most of the engagement variance. A caveat is noted in the report.
- **Tag benchmark quality.** Broad Tumblr tags attract diverse content communities (fan fiction, photography, Sims 4). We apply a minimum 80-word filter to retain only narrative-quality posts for the recommender.
- **Residual temporal confound.** After notes_per_day normalization, year still correlates with the target at r=−0.215 (reduced from −0.612). This reflects genuine audience size changes over the active era.
